<h2>Overview</h2>
<p>
The <strong>Random Linear Classifier</strong> is a simple baseline algorithm used in machine learning for classification tasks.
Instead of learning optimal parameters through training, it generates random weights and uses them to classify input data.
</p>

<p>While not useful for production, it is valuable for:</p>
<ul>
  <li>Understanding linear decision boundaries</li>
  <li>Debugging data pipelines</li>
  <li>Establishing baseline performance</li>
</ul>
