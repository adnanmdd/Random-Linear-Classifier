<h1>Credit Card Fraud Detection</h1>
<h2>Overview</h2>
<p>
<strong>Credit Card Fraud Detection</strong> is a binary classification problem where highly imbalanced datasets (few fraudulent, many legitimate) are analyzed to identify fraudulent transactions.The core challenge was to build a machine learning model that can process highly imbalanced datasets to accurately flag fraudulent activity while minimizing false positives This problem was solved using the Random Linear Classifier algorithm.
</p>

<p>
The number of features were incrased to three unlike the previous problem which had two. In this problem, the plane was used to classify the points.  

<h2>Outcome</h2>
<p>After training and evaluating the Random Linear Classifier on the credit card transaction dataset, the following outcomes were achieved:</p>
<ul>
    <li>The model was able to achieve 100% accuracy.</li>
    <li> Applying normalization to the features helped the model converge faster and improved overall detection accuracy.</li>
</ul>
<p>
    Overall, the implementation demonstrates that a Random Linear Classifier can be an effective solution for credit card fraud detection, providing a balance between performance and computational efficiency. This approach lays the foundation for further enhancements, such as combining with ensemble methods or incorporating real-time transaction monitoring.
</p>
